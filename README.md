реализация класса Logger, обеспечивающего потокобезопасную запись и чтение логов с использованием std::fstream и std::shared_mutex.
Программа моделирует работу нескольких потоков, которые одновременно записывают и читают сообщения из общего лог-файла.
-При создании объекта Logger — файл открывается (RAII).
-При уничтожении объекта — файл закрывается.
-Метод writeLog() — потокобезопасная запись строки в log.txt.
-Метод readLog() — потокобезопасное чтение одной строки из файла.
-Многопоточность реализована через std::thread.
